# Test-projet-Md-

J’ai commencé par une première version de test avec le Soleil et trois planètes : Mercure, Vénus et la Terre.
Chaque planète est représentée par un <div> et stylisée avec du CSS (taille, couleur, forme circulaire). Le Soleil est placé au centre de l’écran avec un dégradé et une lueur pour le rendre plus réaliste.

J’ai utilisé Anime.js pour animer les orbites circulaires des planètes autour du Soleil, ce qui m’a permis d’obtenir un mouvement fluide et continu.

Pour créer l’effet d’espace, j’ai généré des étoiles aléatoires avec JavaScript. Chaque étoile est un petit <div> blanc, positionné de manière aléatoire avec une opacité variable afin de rendre le fond plus vivant.
